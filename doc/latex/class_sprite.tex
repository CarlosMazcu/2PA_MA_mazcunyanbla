\doxysubsection{Detailed Description}
Class representing a sprite entity in the game. 

The \mbox{\hyperlink{class_sprite}{Sprite}} class extends the functionality of the base \mbox{\hyperlink{class_entity}{Entity}} class to represent a sprite entity, which is associated with a texture for rendering.

Draw the \mbox{\hyperlink{class_sprite}{Sprite}}. 

This method draws the \mbox{\hyperlink{class_sprite}{Sprite}} on the screen. It checks if the \mbox{\hyperlink{class_sprite}{Sprite}} is enabled and if a valid texture handle is associated with it. If conditions are met, it applies a transformation matrix and uses it to draw the sprite. If the texture handle is nullptr, an error message is printed.
