\doxysubsection{Detailed Description}
Base class for game entities. 

The \mbox{\hyperlink{class_entity}{Entity}} class represents a basic game entity with properties such as position, scale, rotation, and visibility. Derived classes can inherit from this class to create specific types of entities.
