\doxysubsection{Detailed Description}
Class representing a path entity in the game. 

The \mbox{\hyperlink{class_path}{Path}} class extends the functionality of the base \mbox{\hyperlink{class_entity}{Entity}} class to represent a path entity, which is defined by a series of vertices and can have stroke and fill colors.
